function channelModelClass::absorption ( in  obj,

absorption of sound per m in dB

compute absorption first in dB/Km as per Francois-Garrison equations, then convert to absorption in dB/m, which is what we'll need later. Utilizes salinity, temperature, depth etc. set up previously in the object

Parameters
[in]obj- the object (used to hold ocean parameters)
[in]fHz- frequency in Hz
Return values
result- transmission loss in dB/m
